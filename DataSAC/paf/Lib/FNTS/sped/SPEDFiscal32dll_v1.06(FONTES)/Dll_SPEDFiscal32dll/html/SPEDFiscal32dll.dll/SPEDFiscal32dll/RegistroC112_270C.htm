DT_VCTO : STRING 'Data de vencimento do documento de arrecadação
DT_PGTO : STRING 'Data de pagamento do documento de arrecadação, ou data do vencimento, no caso de ICMS antecipado a recolher.

-2 Data Final Inválida (AAAAMMDD) : ' + DT_VCTO
