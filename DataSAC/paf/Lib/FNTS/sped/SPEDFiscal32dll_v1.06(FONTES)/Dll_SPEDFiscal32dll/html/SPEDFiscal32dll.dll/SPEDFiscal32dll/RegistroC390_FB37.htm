Description
 

Registro C390 - REGISTRO ANALÍTICO DAS NOTAS FISCAIS DE VENDA A CONSUMIDOR (CÓDIGO 02)

Source code
  Function RegistroC390 ( CST_ICMS , CFOP , ALIQ_ICMS , VL_OPR , VL_BC_ICMS , VL_ICMS , VL_RED_BC , COD_OBS : String ) : Integer ; stdcall ; export

Parameters
 
Parameter Description
CST_ICMS : STRING 'Código da Situação Tributária, conforme a Tabela indicada no item 4.3.1
CFOP : STRING 'Código Fiscal de Operação e Prestação
ALIQ_ICMS : STRING 'Alíquota do ICMS
VL_OPR : STRING 'Valor total acumulado das operações correspondentes à combinação de CST_ICMS, CFOP e alíquota do ICMS, incluídas as despesas acessórias e acréscimos.
VL_BC_ICMS : STRING 'Valor acumulado da base de cálculo do ICMS, referente à combinação de CST_ICMS, CFOP, e alíquota do ICMS.
VL_ICMS : STRING 'Valor acumulado do ICMS, referente à combinação de CST_ICMS, CFOP e alíquota do ICMS.
VL_RED_BC : STRING 'Valor não tributado em função da redução da base de cálculo do ICMS, referente à combinação de CST_ICMS, CFOP, e alíquota do ICMS.
COD_OBS : STRING 'Código da observação do lançamento fiscal (campo 02 do Registro 0460)

Return
 
Types Description
INTEGER Caso nenhum erro encontrado retorna RegistroC001 devidamente formatado '|C390|' + CST_ICMS, CFOP, ALIQ_ICMS, VL_OPR, VL_BC_ICMS, VL_ICMS, VL_RED_BC, COD_OBS
'Quebra de sequência conforme Lógica de Ordenação Inter-Registros.
'-1 Só deve haver 1(um) Registro' + TIPO + ' no arquivo do SPED : Função Registro' + TIPO + ' chamada mais de 1(uma) vez
'-2 Numero de CFOP inválido: ' + CFOP
