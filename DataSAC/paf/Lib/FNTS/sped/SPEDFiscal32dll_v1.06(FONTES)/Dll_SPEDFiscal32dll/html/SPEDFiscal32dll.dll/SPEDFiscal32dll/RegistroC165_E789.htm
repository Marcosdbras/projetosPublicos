REGISTRO C165 - OPERAÇÕES COM COMBUSTÍVEIS(CÓDIGO 01; 55)

Source code
  Function RegistroC165 ( COD_PART , VEIC_ID , COD_AUT , NR_PASSE , HORA , TEMPER , QTD_VOL , PESO_BRT , PESO_LIQ , NOM_MOT , CPF , UF_ID : String ) : Integer ; stdcall ; export

Parameters
 
Parameter Description
COD_PART : STRING 'Código do participante (campo 02 do Registro 0150):
VEIC_ID : STRING 'Placa de identificação do veículo
COD_AUT : STRING 'Código da autorização fornecido pela SEFAZ (combustíveis)
NR_PASSE : STRING 'Número do Passe Fiscal
HORA : STRING 'Hora da saída das mercadorias
TEMPER : STRING 'Temperatura em graus Celsius utilizada para quantificação do volume de combustível
QTD_VOL : STRING 'Quantidade de volumes transportados
PESO_BRT : STRING 'Peso bruto dos volumes transportados (em Kg)
PESO_LIQ : STRING 'Peso líquido dos volumes transportados (em Kg)
NOM_MOT : STRING 'Nome do motorista
CPF : STRING 'CPF do motorista
UF_ID : STRING 'Sigla da UF da placa do veículo

Return
 
Types Description
INTEGER Caso nenhum erro encontrado retorna RegistroC001 devidamente formatado '|C165|' + COD_PART, VEIC_ID, COD_AUT, NR_PASSE, HORA, TEMPER,QTD_VOL, PESO_BRT, PESO_LIQ, NOM_MOT, CPF, UF_ID
'Quebra de sequência conforme Lógica de Ordenação Inter-Registros.
-1 CPF do motorista incorreto : ' + CPF
-2 Sigla da UF da placa do veículo incorreta : ' + UF_ID
