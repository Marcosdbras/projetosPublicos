Registro 0100 - DADOS DO CONTABILISTA

Source code
  Function Registro0100 ( NOME , CPF , CRC , CNPJ , CEP , ENDERECO , NUM , COMPL , BAIRRO , FONE , FAX , EMAIL , COD_MUN : String ) : Integer ; stdcall ; export

Parameters
 
Parameter Description
NOME : STRING Nome do contabilista/escritório:
CPF : STRING Número de inscrição no CPF:
CRC : STRING Número de inscrição no Conselho Regional:
CNPJ : STRING CNPJ do escritório de contabilidade, se houver:
CEP : STRING Código de Endereçamento Postal:
ENDERECO : STRING Logradouro e endereço do imóvel:
NUM : STRING Número do imóvel:
COMPL : STRING Dados complementares do endereço:
BAIRRO : STRING Bairro em que o imóvel está situado:
FONE : STRING Número do telefone:
FAX : STRING Número do fax:
EMAIL : STRING Endereço do correio eletrônico:
COD_MUN : STRING Código do município, conforme tabela IBGE:

Return
 
Types Description
INTEGER Caso nenhum erro encontrado retorna Registro0100 devidamente formatado '|0100|' + NOME + CPF + CRC + CNPJ + CEP + ENDERECO + NUM + COMPL + BAIRRO + FONE + FAX + EMAIL
'Quebra de sequência conforme Lógica de Ordenação Inter-Registros.'
-1 Só deve haver 1(um) Registro' + TIPO + ' no arquivo do SPED : Função Registro' + TIPO + ' chamada mais de 1(uma) vez'
-2 CNPJ ou CPF Inválido : ' + CNPJ
-3 Número de inscrição no CPF Inválida: ' + CPF
-4 Nome do Contribuinte Inválido: Preenchimento Obrigatório'
-5 Código de Endereçamento Postal Inválida: ' + CEP
