Registro E210 - APURAÇÃO DO ICMS - SUBSTITUIÇÃO TRIBUTÁRIA

Source code
  Function RegistroE210 ( IND_MOV_ST , VL_SLD_CRED_ANT_ST , VL_DEVOL_ST , VL_RESSARC_ST , VL_OUT_CRED_ST , VL_AJ_CREDITOS_ST , VL_RETENCAO_ST , VL_OUT_DEB_ST , VL_AJ_DEBITOS_ST , VL_SLD_DEV_ANT_ST , VL_DEDUCOES_ST , VL_ICMS_RECOL_ST , VL_SLD_CRED_ST_TRANSPORTAR , DEB_ESP_ST : String ) : Integer ; stdcall ; export

Parameters
 
Parameter Description
IND_MOV_ST : STRING 'Indicador de movimento: 0 - Sem operações com ST; 1 - Com operações de ST
VL_SLD_CRED_ANT_ST : STRING 'Valor do "Saldo credor de período anterior - Substituição Tributária"
VL_DEVOL_ST : STRING 'Valor total do ICMS ST de devolução de mercadorias
VL_RESSARC_ST : STRING 'Valor total do ICMS ST de ressarcimentos
VL_OUT_CRED_ST : STRING 'Valor total de Ajustes "Outros créditos ST" e "Estorno de débitos ST"
VL_AJ_CREDITOS_ST : STRING 'Valor total dos ajustes a crédito de ICMS ST, provenientes de ajustes do documento fiscal.
VL_RETENCAO_ST : STRING 'Valor Total do ICMS retido por Substituição Tributária
VL_OUT_DEB_ST : STRING 'Valor Total dos ajustes "Outros débitos ST" e "Estorno de créditos ST"
VL_AJ_DEBITOS_ST : STRING 'Valor total dos ajustes a débito de ICMS ST, provenientes de ajustes do documento fiscal.
VL_SLD_DEV_ANT_ST : STRING 'Valor total de "Saldo devedor antes das deduções" = (08+09+10)-(03+04+05+06+07)]
VL_DEDUCOES_ST : STRING 'Valor total dos ajustes "Deduções ST"
VL_ICMS_RECOL_ST : STRING 'Imposto a recolher ST (11-12)
VL_SLD_CRED_ST_TRANSPORTAR : STRING 'Saldo credor de ST a transportar para o período seguinte [(03+04+05+06+07)– (08+09+10)].
DEB_ESP_ST : STRING 'Valores recolhidos ou a recolher, extra-apuração.

'Quebra de sequência conforme Lógica de Ordenação Inter-Registros.
