Registro 0150 - TABELA DE CADASTRO DO PARTICIPANTE

Source code
  Function Registro0150 ( COD_PART , NOME , COD_PAIS , CNPJ , CPF , IE , COD_MUN , SUFRAMA , ENDERECO , NUM , COMPL , BAIRRO : String ) : Integer ; stdcall ; export

Parameters
 
Parameter Description
COD_PART : STRING Código de identificação do participante:
NOME : STRING Nome pessoal ou empresarial:
COD_PAIS : STRING Código do país do participante:
CNPJ : STRING CNPJ do participante:
CPF : STRING CPF do participante na unidade da federação do destinatário:
IE : STRING Inscrição Estadual do participante:
COD_MUN : STRING Código do município:
SUFRAMA : STRING Número de inscrição na Suframa:
ENDERECO : STRING Logradouro e endereço do imóvel:
NUM : STRING Número do imóvel:
COMPL : STRING Dados complementares do endereço:
BAIRRO : STRING Bairro em que o imóvel está situado:

Return
 
Types Description
INTEGER Caso nenhum erro encontrado retorna Registro0150 devidamente formatado '|0150|' + COD_PART + NOME + COD_PAIS + CNPJ + CPF + IE + COD_MUN + SUFRAMA + ENDERECO + NUM + COMPL + BAIRRO
Quebra de sequência conforme Lógica de Ordenação Inter-Registros.
-1 CNPJ Inválido : ' + CNPJ
-2 CPF Inválido : ' + CPF
-3 Código do município do domicílio fiscal : ' + COD_MUN
-4 Código do município do domicílio fiscal : ' + COD_PAIS
