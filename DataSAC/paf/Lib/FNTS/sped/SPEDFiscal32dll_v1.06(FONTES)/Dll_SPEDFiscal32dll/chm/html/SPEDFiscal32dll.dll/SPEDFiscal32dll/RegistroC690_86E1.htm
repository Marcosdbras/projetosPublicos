Name
  RegistroC690

Parent
  SPEDFiscal32dll

Visibility
  PUBLIC

Description
 

Registro C690 - REGISTRO ANALÍTICO DOS DOCUMENTOS (Notas Fiscais/Contas de Energia Elétrica (Código 06), Nota Fiscal/Conta de Fornecimento d´água CANALIZADA (código 29) e Nota Fiscal/Conta de Fornecimento de Gás (Código 28)

Source code
  Function RegistroC690 ( CST_ICMS , CFOP , ALIQ_ICMS , VL_OPR , VL_BC_ICMS , VL_ICMS , VL_RED_BC , VL_BC_ICMS_ST , VL_ICMS_ST , COD_OBS : String ) : Integer ; stdcall ; export

Parameters
 
Parameter Description
CST_ICMS : STRING 'Código da Situação Tributária, conforme a tabela indicada no item 4.3.1
CFOP : STRING 'Código Fiscal de Operação e Prestação, conforme a tabela indicada no item 4.2.2
ALIQ_ICMS : STRING 'Alíquota do ICMS
VL_OPR : STRING 'Valor da operação correspondente à combinação de CST_ICMS, CFOP, e alíquota do ICMS.
VL_BC_ICMS : STRING 'Parcela correspondente ao "Valor da base de cálculo do ICMS" referente à combinação CST_ICMS, CFOP e alíquota do ICMS
VL_ICMS : STRING 'Parcela correspondente ao "Valor do ICMS" referente à combinação CST_ICMS, CFOP e alíquota do ICMS
VL_RED_BC : STRING 'Valor não tributado em função da redução da base de cálculo do ICMS, referente à combinação de CST_ICMS, CFOP e alíquota do ICMS.
VL_BC_ICMS_ST : STRING 'Valor da base de cálculo do ICMS substituição tributária
VL_ICMS_ST : STRING 'Valor do ICMS retido por substituição tributária
COD_OBS : STRING 'Código da observação do lançamento fiscal (campo 02 do Registro 0460)

Return
 
Types Description
INTEGER Caso nenhum erro encontrado retorna RegistroC001 devidamente formatado '|C690|' + CST_ICMS, CFOP, ALIQ_ICMS, VL_OPR, VL_BC_ICMS, VL_ICMS, VL_RED_BC, VL_BC_ICMS_ST, VL_ICMS_ST, COD_OBS
'Quebra de sequência conforme Lógica de Ordenação Inter-Registros.
-1 Numero de CFOP inválido: ' + CFOP
