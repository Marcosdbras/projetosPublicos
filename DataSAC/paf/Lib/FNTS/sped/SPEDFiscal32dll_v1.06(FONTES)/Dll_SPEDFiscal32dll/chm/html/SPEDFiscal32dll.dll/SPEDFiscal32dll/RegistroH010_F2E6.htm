Registro H010 - INVENTÁRIO

Source code
  Function RegistroH010 ( COD_ITEM , UNID , QTD , VL_UNIT , VL_ITEM , IND_PROP , COD_PART , TXT_COMP , COD_CTA : String ) : Integer ; stdcall ; export

Parameters
 
Parameter Description
COD_ITEM : STRING 'Código do item (campo 02 do Registro 0200)
UNID : STRING 'Unidade do item
QTD : STRING 'Quantidade do item
VL_UNIT : STRING 'Valor unitário do item
VL_ITEM : STRING 'Valor do item
IND_PROP : STRING 'Indicador de propriedade/posse do item: 0- Item de propriedade do informante e em seu poder; 1- Item de propriedade do informante em posse de terceiros; 2- Item de propriedade de terceiros em posse do informante
COD_PART : STRING 'Código do participante (campo 02 do Registro 0150): proprietário/possuidor que não seja o informante do arquivo
TXT_COMP : STRING 'Descrição complementar.
COD_CTA : STRING 'Código da conta analítica contábil debitada/creditada

Return
 
Types Description
INTEGER Caso nenhum erro encontrado retorna RegistroH010 devidamente formatado |H010| + COD_ITEM + UNID + QTD + VL_UNIT + VL_ITEM + IND_PROP + COD_PART + TXT_COMP + COD_CTA
'Quebra de sequência conforme Lógica de Ordenação Inter-Registros.
