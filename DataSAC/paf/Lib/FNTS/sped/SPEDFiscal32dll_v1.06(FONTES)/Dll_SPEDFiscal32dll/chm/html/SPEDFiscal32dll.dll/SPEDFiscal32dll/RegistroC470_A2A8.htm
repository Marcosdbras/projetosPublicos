Parameters

COD_ITEM : STRING 'Código do item (campo 02 do Registro 0200)

CST_ICMS : STRING 'Código da Situação Tributária, conforme a Tabela indicada no item 4.3.1.
CFOP : STRING 'Código Fiscal de Operação e Prestação
ALIQ_ICMS : STRING 'Alíquota do ICMS - Carga tributária efetiva em percentual
VL_PIS : STRING 'Valor do PIS
VL_COFINS : STRING 'Valor da COFINS

Return
