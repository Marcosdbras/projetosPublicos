Registro E530 - AJUSTES DA APURAÇÃO DO IPI

Source code
  Function RegistroE530 ( IND_AJ , VL_AJ , COD_AJ , IND_DOC , NUM_DOC , DESCR_AJ : String ) : Integer ; stdcall ; export

Parameters
 
Parameter Description
IND_AJ : STRING 'Indicador do tipo de ajuste: 0- Ajuste a débito; 1- Ajuste a crédito
VL_AJ : STRING 'Valor do ajuste
COD_AJ : STRING 'Código do ajuste da apuração, conforme a Tabela indicada no item 4.5.4.
IND_DOC : STRING 'Indicador da origem do documento vinculado ao ajuste: 0 - Processo Judicial; 1 - Processo Administrativo; 2 - PER/DCOMP; 9 - Outros.
NUM_DOC : STRING 'Número do documento / processo / declaração ao qual o ajuste está vinculado, se houver
DESCR_AJ : STRING 'Descrição resumida do ajuste.

Return
 
Types Description
INTEGER Caso nenhum erro encontrado retorna RegistroE530 devidamente formatado |E530| + IND_AJ + VL_AJ + COD_AJ + IND_DOC + NUM_DOC + DESCR_AJ
'Quebra de sequência conforme Lógica de Ordenação Inter-Registros.
