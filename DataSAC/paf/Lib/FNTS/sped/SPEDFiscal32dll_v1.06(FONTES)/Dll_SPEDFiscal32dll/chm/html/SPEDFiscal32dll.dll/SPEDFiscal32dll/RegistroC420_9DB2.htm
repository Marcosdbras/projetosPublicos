Registro C420 - REGISTRO DOS TOTALIZADORES PARCIAIS DA REDUÇÃO Z (COD 02 e 2D)

Source code
  Function RegistroC420 ( COD_TOT_PAR , VLR_ACUM_TO , NR_TOT , DESCR_NR_TOT : String ) : Integer ; stdcall ; export

Parameters
 
Parameter Description
COD_TOT_PAR : STRING 'Código do totalizador, conforme Tabela 4.4.6
VLR_ACUM_TO : STRING 'Valor acumulado no totalizador, relativo à respectiva Redução Z.
NR_TOT : STRING 'Número do totalizador quando ocorrer mais de uma situação com a mesma carga tributária efetiva.
DESCR_NR_TOT : STRING 'Descrição da situação tributária relativa ao totalizador parcial, quando houver mais de um com a mesma carga tributária efetiva.

Return
 
Types Description
INTEGER Caso nenhum erro encontrado retorna RegistroC420 devidamente formatado |C420| + COD_TOT_PAR + VLR_ACUM_TO + NR_TOT + DESCR_NR_TOT
'Quebra de sequência conforme Lógica de Ordenação Inter-Registros.
