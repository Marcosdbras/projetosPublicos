Name
  RegistroD100

Parent
  SPEDFiscal32dll

Visibility
  PUBLIC

Description
 

Registro D100 - NOTA FISCAL DE SERVIÇO DE TRANSPORTE (CÓDIGO 07) E CONHECIMENTOS DE TRANSPORTE RODOVIÁRIO DE CARGAS (CÓDIGO 08), CONHECIMENTOS DE TRANSPORTE DE CARGAS AVULSO (CÓDIGO 8B), AQUAVIÁRIO DE CARGAS (CÓDIGO 09), AÉREO (CÓDIGO 10), FERROVIÁRIO DE CARGAS (CÓDIGO 11) E MULTIMODAL DE CARGAS (CÓDIGO 26), NOTA FISCAL DE TRANSPORTE FERROVIÁRIO DE CARGA ( CÓDIGO 27) E CONHECIMENTO DE TRANSPORTE ELETRÔNICO – CT-e (CÓDIGO 57)

Source code
  Function RegistroD100 ( IND_OPER , IND_EMIT , COD_PART , COD_MOD , COD_SIT , SER , SUB , NUM_DOC , CHV_CTE , DT_DOC , DT_A_P , TP_CT_E , CHV_CTE_REF , VL_DOC , VL_DESC , IND_FRT , VL_SERV , VL_BC_ICMS , VL_ICMS , VL_NT , COD_INF , COD_CTA : String ) : Integer ; stdcall ; export

Parameters
 
Parameter Description
IND_OPER : STRING 'Indicador do tipo de operação: 0- Aquisição; 1- Prestação
IND_EMIT : STRING 'Indicador do emitente do documento fiscal: 0- Emissão própria; 1- Terceiros
COD_PART : STRING 'Código do participante (campo 02 do Registro 0150): - do prestador de serviço, no caso de aquisição de serviço; - do tomador do serviço, no caso de prestação de serviços.
COD_MOD : STRING 'Código do modelo do documento fiscal, conforme a Tabela 4.1.1
COD_SIT : STRING 'Código da situação do documento fiscal, conforme a Tabela 4.1.2
SER : STRING 'Série do documento fiscal
SUB : STRING 'Subsérie do documento fiscal
NUM_DOC : STRING 'Número do documento fiscal
CHV_CTE : STRING 'Chave do Conhecimento de Transporte Eletrônico
DT_DOC : STRING 'Data da emissão do documento fiscal
DT_A_P : STRING 'Data da aquisição ou da prestação do serviço
TP_CT_E : STRING 'Tipo de Conhecimento de Transporte Eletrônico conforme definido no Manual de Integração do CT-e
CHV_CTE_REF : STRING 'Chave do CT-e de referência cujos valores foram complementados (opção “1” do campo anterior) ou cujo débito foi anulado(opção “2” do campo anterior).
VL_DOC : STRING 'Valor total do documento fiscal
VL_DESC : STRING 'Valor total do desconto
IND_FRT : STRING 'Indicador do tipo do frete: 0- Por conta de terceiros; 1- Por conta do emitente; 2- Por conta do destinatário; 9- Sem frete
VL_SERV : STRING 'Valor total da prestação de serviço
VL_BC_ICMS : STRING 'Valor da base de cálculo do ICMS
VL_ICMS : STRING 'Valor do ICMS
VL_NT : STRING 'Valor não-tributado
COD_INF : STRING 'Código da informação complementar do documento fiscal (campo 02 do Registro 0450)
COD_CTA : STRING 'Código da conta analítica contábil debitada/creditada

Return
 
Types Description
INTEGER Caso nenhum erro encontrado retorna RegistroD100 devidamente formatado |D100| + IND_OPER + IND_EMIT + COD_PART + COD_MOD + COD_SIT + SER + SUB + NUM_DOC + CHV_CTE + DT_DOC + DT_A_P + TP_CT_E + CHV_CTE_REF + VL_DOC + VL_DESC + IND_FRT + VL_SERV + VL_BC_ICMS + VL_ICMS + VL_NT + COD_INF + COD_CTA
'Quebra de sequência conforme Lógica de Ordenação Inter-Registros.
'-1 Data Inicial Inválida (DDMMAAAA) :: ' + DT_DOC
'-2 Data Final Inválida (DDMMAAAA) :: ' + DT_A_P
