  Function RegistroD160 ( DESPACHO , CNPJ_CPF_REM , IE_REM , COD_MUN_ORI , CNPJ_CPF_DEST , IE_DEST , COD_MUN_DEST : String ) : Integer ; stdcall ; export

Parameters
 
Parameter Description
DESPACHO : STRING 'Identificação do número do despacho
CNPJ_CPF_REM : STRING 'CNPJ ou CPF do remetente das mercadorias que constam na nota fiscal.
IE_REM : STRING 'Inscrição Estadual do remetente das mercadorias que constam na nota fiscal.
COD_MUN_ORI : STRING 'Código do Município de origem, conforme tabela IBGE
CNPJ_CPF_DEST : STRING 'CNPJ ou CPF do destinatário das mercadorias que constam na nota fiscal.
IE_DEST : STRING 'Inscrição Estadual do destinatário das mercadorias que constam na nota fiscal.
COD_MUN_DEST : STRING 'Código do Município de destino, conforme tabela IBGE

Return
 
Types Description
INTEGER Caso nenhum erro encontrado retorna RegistroD160 devidamente formatado |D160| + DESPACHO + CNPJ_CPF_REM + IE_REM + COD_MUN_ORI + CNPJ_CPF_DEST + IE_DEST + COD_MUN_DEST
