Registro 1110: OPERAÇÕES DE EXPORTAÇÃO INDIRETA DE PRODUTOS NÃO INDUSTRIALIZADOS PELO ESTABELECIMENTO EMITENTE. ---

Source code
  Function Registro1110 ( COD_PART , COD_MOD , SER , NUM_DOC , DT_DOC , CHV_NFE , NR_ , QTD , UNID : String ) : Integer ; stdcall ; export

Parameters
 
Parameter Description
COD_PART : STRING 'Código do participante-Fornecedor da Mercadoria destinada à exportação (campo 02 do Registro 0150)
COD_MOD : STRING 'Código do documento fiscal, conforme a Tabela 4.1.1
SER : STRING 'Série do documento fiscal recebido com fins específicos de exportação.
NUM_DOC : STRING 'Número do documento fiscal recebido com fins específicos de exportação.
DT_DOC : STRING 'Data da emissão do documento fiscal recebido com fins específicos de exportação
CHV_NFE : STRING 'Chave da Nota Fiscal Eletrônica
NR_ : STRING 'Número do Memorando de Exportação
QTD : STRING 'Quantidade do item efetivamente exportado.
UNID : STRING 'Unidade do item (Campo 02 do registro 0190)

Return
 
Types Description
INTEGER Caso nenhum erro encontrado retorna RegistroC001 devidamente formatado '|1110|' + COD_PART, COD_MOD, SER, NUM_DOC, DT_DOC, CHV_NFE, NR_, QTD, UNID
'Quebra de sequência conforme Lógica de Ordenação Inter-Registros.
-1 Data Final Inválida (AAAAMMDD) : ' + DT_DOC
