Name
  RegistroD300

Parent
  SPEDFiscal32dll

Visibility
  PUBLIC

Description
 

Registro D300 - REGISTRO ANALÍTICO DOS BILHETES CONSOLIDADOS DE PASSAGEM RODOVIÁRIO (CÓDIGO 13), DE PASSAGEM AQUAVIÁRIO (CÓDIGO 14), DE PASSAGEM E NOTA DE BAGAGEM (CÓDIGO 15) E DE PASSAGEM FERROVIÁRIO (CÓDIGO 16)

Source code
  Function RegistroD300 ( COD_MOD , SER , SUB , NUM_DOC_INI , NUM_DOC_FIN , CST_ICMS , CFOP , ALIQ_ICMS , DT_DOC , VL_OPR , VL_DESC , VL_SERV , VL_SEG , VL_OUT_DESP , VL_BC_ICMS , VL_ICMS , VL_RED_BC , COD_OBS , COD_CTA : String ) : Integer ; stdcall ; export

Parameters
 
Parameter Description
COD_MOD : STRING 'Código do modelo do documento fiscal, conforme a Tabela 4.1.1
SER : STRING 'Série do documento fiscal
SUB : STRING 'Subsérie do documento fiscal
NUM_DOC_INI : STRING 'Número do primeiro documento fiscal emitido (mesmo modelo, série e subsérie)
NUM_DOC_FIN : STRING 'Número do último documento fiscal emitido (mesmo modelo, série e subsérie)
CST_ICMS : STRING 'Código da Situação Tributária, conforme a Tabela indicada no item 4.3.1
CFOP : STRING 'Código Fiscal de Operação e Prestação conforme tabela indicada no item 4.2.2
ALIQ_ICMS : STRING 'Alíquota do ICMS
DT_DOC : STRING 'Data da emissão dos documentos fiscais
VL_OPR : STRING 'Valor total acumulado das operações correspondentes à combinação de CST_ICMS, CFOP e alíquota do ICMS, incluídas as despesas acessórias e acréscimos.
VL_DESC : STRING 'Valor total dos descontos
VL_SERV : STRING 'Valor total da prestação de serviço
VL_SEG : STRING 'Valor de seguro
VL_OUT_DESP : STRING 'Valor de outras despesas
VL_BC_ICMS : STRING 'Valor total da base de cálculo do ICMS
VL_ICMS : STRING 'Valor total do ICMS
VL_RED_BC : STRING 'Valor não tributado em função da redução da base de cálculo do ICMS, referente à combinação de CST_ICMS, CFOP e alíquota do ICMS.
COD_OBS : STRING 'Código da observação do lançamento fiscal (campo 02 do Registro 0460)
COD_CTA : STRING 'Código da conta analítica contábil debitada/creditada

Return
 
Types Description
INTEGER Caso nenhum erro encontrado retorna RegistroD300 devidamente formatado |D300| + COD_MOD + SER + SUB + NUM_DOC_INI + NUM_DOC_FIN + CST_ICMS + CFOP + ALIQ_ICMS + DT_DOC + VL_OPR + VL_DESC + VL_SERV + VL_SEG + VL_OUT_DESP + VL_BC_ICMS + VL_ICMS + VL_RED_BC + COD_OBS + COD_CTA
'Quebra de sequência conforme Lógica de Ordenação Inter-Registros.
