Data de liberação: 14/10/2011

Versão Arquivo: 1.5.6

Correções / Implementações:

- Tratamento na geração dos arquivos NFP e NFPTDM para evitar sobreposição destes quando gerados simultaneamente no mesmo diretório.

- Tela de informações (Sobre) com dados de versão do programa e links para acesso ao site Desenvolvedores Daruma e ao vídeo do GAD no Daruma Developers Channel9 (youtube)

- Modificações na arquitetura usando Threads para melhorar desempenho.

- Tela de parametrização da divisão dos arquvivos.

- Carregamento automático dos dados do xml na tela de parametrização do SINTEGRA.

- Campos para informação dos intervalos mudam de acordo com o tipo do intervalo seleciondo. Quando tipo do intervalo for COO ou CRZ é disponibilizado edits (caracteres apenas numéricos) e no se o tipo for data disponibiliza um calendário.

- Tratamento de limite para o número máximo de COO e CRZ.

- Calendário para seleção dos intervalos de data limitados de acordo com o ultimo movimento do ECF.

- Opção para abrir o diretório onde os arquivos estão sendo gerados.

- Tela de "Status da Geração" em modal e sem opção de fechamento antes de finalizado todo o processo de geração dos arquivos.

- Novo diálogo para seleção de diretórios para selecionar o local de geração dos arquivos.

- Informações do ECF atualizados automaticamente ao final de cada processo de geração dos arquivos.

- Botão para mostrar/ocultar informações do ECF com efeito Downed.
